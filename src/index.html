Styling der Komponente

Globale Definitionen um alle Komponenten identisch zu stylen

  /* <style> set all speaker at page to defaults of component via :root */
    :root {
      --speaker-background: transparent;
      --speaker-color: blue;
      --speaker-padding: 5px;
      --speaker-font-size: medium;
      --speaker-border: 0;
      --speaker-width: 36px;
      --speaker-height: 36px;
    }

  or via part

    /* <style> set all speaker at page to defaults of component via ::part */
    ::part(speakerpane) {
      --speaker-background: transparent;
      --speaker-color: blue;
      --speaker-padding: 5px;
      --speaker-font-size: medium;
      --speaker-border: 0;
      --speaker-width: 36px;
      --speaker-height: 36px;
    }
